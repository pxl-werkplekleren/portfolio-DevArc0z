Introductie portfolio

Ik ben Meical. Ik ben een gemotiveerde en nieuwsgierige persoon met een sterke interesse in mijn vakgebied. In mijn portfolio laat ik zien hoe ik mij de afgelopen tijd heb ontwikkeld. Het bevat een selectie van projecten waarin mijn vaardigheden, creativiteit duidelijk naar voren komen, maar denk process wat minder. Elk project laat zien hoe chaotisch mijn denk process is. Daarnaast reflecteer ik op mijn leerproces en benoem ik wat ik heb geleerd en hoe ik iets beter kan proberen. Mijn portfolio geeft niet alleen het eindresultaat weer, maar ook andere zaken. Ik vind het belangrijk om mezelf te blijven uitdagen en nieuwe dingen te leren. Met dit portfolio wil ik een duidelijk en eerlijk beeld geven van mijn wat ik wel en niet kan.
